# My React Boilerplate (ServerSideRendering)

[![Deploy](https://www.herokucdn.com/deploy/button.png)](https://heroku.com/deploy?template=https://github.com/koba04/react-boilerplate)

* react
  * react-router-component
* express
  * express-react-views
* backbone
* browserify
  * watchify
  * reactify
  * stylify
* insert-css

## setup
```
npm install
```

## watch and autobuild
```
npm run dev
```

## run server
```
node app.js
```

## build
* create bundle.js
```
npm run build
```
