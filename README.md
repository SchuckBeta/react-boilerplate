# My React Boilerplate

* react
  * react-router
* backbone
* browserify
  * beefy (auto build & livereload)
  * reactify
  * stylify
* insert-css

## setup
```
npm install
```

## start develop
* livereload and autobuild
  * not created bundle.js
```
npm run dev
```

## build
* create bundle.js
```
npm run build
```
